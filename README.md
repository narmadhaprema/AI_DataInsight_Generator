# AI Data Insight Generator

An **AI-powered Streamlit web app** that automatically analyzes uploaded datasets, generates insights, builds simple ML models, and visualizes patterns â€” all in one click.

---

## Features
- Upload any CSV file and explore data instantly  
- Automatic statistical summary and missing value analysis  
- Visualizations (histograms, correlation heatmaps, boxplots)  
- Optional AI-powered data insights using OpenAI GPT  
- AutoML for regression/classification using Scikit-learn  
- Streamlit UI for interactive use and easy deployment
